188 /*================== Thermal Runaway Protection ==============================
189 This is a feature to protect your printer from burn up in flames if it has
190 a thermistor coming off place (this happened to a friend of mine recently and
191 motivated me writing this feature).
192 
193 The issue: If a thermistor come off, it will read a lower temperature than actual.
194 The system will turn the heater on forever, burning up the filament and anything
195 else around.
196 
197 After the temperature reaches the target for the first time, this feature will
198 start measuring for how long the current temperature stays below the target
199 minus _HYSTERESIS (set_temperature - THERMAL_RUNAWAY_PROTECTION_HYSTERESIS).
200 
201 If it stays longer than _PERIOD, it means the thermistor temperature
202 cannot catch up with the target, so something *may be* wrong. Then, to be on the
203 safe side, the system will he halt.
204 
205 Bear in mind the count down will just start AFTER the first time the
206 thermistor temperature is over the target, so you will have no problem if
207 your extruder heater takes 2 minutes to hit the target on heating.
208 
209 */
